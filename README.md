# 📅 Interactive Wall Calendar (React)

A visually rich and interactive **wall calendar component** built using React, inspired by a real-world hanging calendar design. This project combines **UI aesthetics + functional features** like date range selection and notes.

---

## ✨ Features

* 🧱 **Wall Calendar Design**

  * Realistic hanging effect (hook, strings, spiral binding)
  * Hero image with month & year overlay

* 📅 **Date Range Selection**

  * Select start and end dates
  * Highlight selected range clearly

* 📝 **Notes Section**

  * Add personal notes


* 📱 **Responsive Layout**

  * Works on desktop and mobile
  * Clean and compact UI

* 🎨 **Modern UI**

  * Built with Tailwind CSS
  * Smooth hover effects
  * Clean typography and spacing

---

## 🛠️ Tech Stack

* **React (Vite)**
* **Tailwind CSS**
* **JavaScript (ES6+)**


---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/interactive-calendar.git
cd interactive-calendar
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project

```bash
npm run dev
```
